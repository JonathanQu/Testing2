HOW TO INSTALL THE STARTUP WEBSITES
1.Hit your windows key and r at the same time
2.Inside the Run Prompt, type 
C:\Users\ YOUR USERNAME \AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup
where username is your current username for your windows account on the computer.
3.Drag the contents of the ToStartup folder into
C:\Users\username\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup
4.Within the websites .txt, type the full webaddress of the website you want open on startup, but there
may only be one website per line

Note,, the web browser used will be your default webrowser
