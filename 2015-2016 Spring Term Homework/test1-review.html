Here are some terms to review for terminal

Basic commands taught in Dyrland-Weaver's Ckass
Command Example Description
~ cd ~/ shortcut from home directory
.. cd .. navigates toward parent directory
cd cd Desktop cd tells terminal to navigate to whatever is specified, which in this case, the Desktop
pwd pwd outputs the location of the current directory terminal is in
man man cd shows documentation of a certain command, which is the cd command in this example
ls ls -all lists all the files/directories in the current directory, the -all or -a tag displays hidden things as well
cat cat swag.txt will display text from a txt file, which in this case is called swag.txt
ssh ssh homer.stuy.edu will connect to homer.stuy.edu's remote control server
./ ./starbound.exe will run the program starbound.exe
touch touch hype.txt will create an empty text file called hype.txt
mkdir mkdir music creates a directory called music
ping ping homer.stuy.edu sends data packets to the homer.stuy.edu server and recieves those packets back while measuring the time it takes
traceroute traceroute homer.stuy.edu shows the path a data packet travels through to reach homer.stuy.edu
